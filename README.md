* ls -オプション　パス　一覧表示する -aで不可視ファイルを表示 -lはタイプ権限、所有者や所有グループなどの詳細情報を表示する
* pwd(Print Working Directory) 現在自分が位置しているディレクトリ＝ワーキングディレクトリがどこであるか出力してくれる。
* cd(Change Directory) ワーキングディレクトリを指定したディレクトリに変更するコマンド。指定をすることで特定のディレクトリをに移動することができる。
  * ~ホームディレクトリ /ルートディレクトリ -移動前に位置していたディレクトリ .ワーキングディレクトリ ..ワーキングディレクトリの一階層上のディレクトリ ../../ワーキングディレクトリの二階層上のディレクトリ
* 絶対パス /を起点として移動できる 相対パス ワーキングディレクトリから参照する移動方法
* mkdir ディレクトリ名 ディレクトリを作成する mkdir -p test2/test3と記述すると間のディレクトリも作成してくれる。
* touch ファイル名 ファイルを作成するコマンド
* && AND関数で、左辺の実行が成功すれば、右辺を実行する 例 $ cd /home/test1 && ls 左辺の移動が成功するとls一覧表示される
* cp コピー元 コピー先 ファイル・ディレクトリのコピーを行うコマンド　-rオプションを指定することでディレクトリの複製ができる。
* mv 変更元 変更先 ファイルやディレクトリの移動を行うコマンド。元のファイル名とは違うファイル名をしているすることで、移動と同時にファイル名の変更を行うこともできます。
* rm 削除するファイルorディレクトリ ファイルやディレクトリを削除するコマンド -rを使用することで、ディレクトリの削除も行えます。rm -rf/というコマンドは全て削除されてしまうので、絶対に使用しないこと
* ＊ ワイルドカード ０文字以上の任意の文字 ＊_bk.phpと指定した場合 ＊が何文字あったとしても対象とする
* ？ 任意の１文字を表す記号 linux_practice？と指定した場合、linux_practice3を対象とするがlinux_practice30は対象とならない。
* \>　左辺の実行結果の出力を右辺で指定したファイルに書き込む記号。echo '# Hello World' > README.md と記述するとREADME.mdファイルに文章が書き込まれる。しかし、既存のファイル名を指定した場合は中身が上書きされる。
* \>> 左辺の実行出力結果を右辺で指定したファイルの末尾に書き込む記号。ファイルが存在しない場合新規作成となる。中身は上書きされず追加される。
* cat 指定したファイルの内容をターミナル上に出力するコマンド
* vim ターミナル上で起動できるエディタ。vim ファイル名で実行することでその場でファイルの編集・保存が可能。
  * ノーマルモード 移動・保存・検索・削除などの操作をキーで行うことができる escキー
  * j 下に移動 k 上に移動 h左に移動 l右に移動 :w 保存 :wq保存して終了 :q!保存せずに強制終了 u変更内容を一つもとに戻す Ctrl+r 変更内容を一つ前に進める
  * インサートモード ファイル編集モード入力したキーがそのままファイルに反映される iキー
